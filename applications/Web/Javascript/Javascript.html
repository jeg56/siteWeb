JavaScript

JavaScript permet de rendre les pages HTML plus dynamiques et interactives.
Balises <script>
La balise <script> permet de définir un script côté client (JavaScript).
L'élément <script> contient des instructions ou pointe vers un fichier de script externe via l'attribut src.
Les utilisations courantes de JavaScript sont la manipulation d'images, la validation de formulaires et les modifications dynamiques de contenu.
Pour sélectionner un élément HTML, JavaScript utilise souvent la méthode document.getElementById (id).
