Definition d'une class 

class Personne: 
   """Classe définissant une personne caractérisée par :
   - son nom
   - son salaire"""

   nb_cree = 0 # Le compteur vaut 0 au départ 

    /*** Notre méthode constructeur ***/
   def __init__(self):
      """Définition d'1 attribut"""
      self.nom = "Dupont"
      self._dictionnaire = {}
      self._salaire=0
      Personne.nb_cree += 1

    /*** format d'une méthode spéciale : __mamethode__ ***/
    def __str__(self):
        """Méthode permettant d'afficher plus joliment notre objet"""
        return "{} gagne {} €".format(self.nom, self._salaire)

    def __getattr__(self, nom):
       """Si Python ne trouve pas l'attribut nommé nom, il appelle cette méthode. 
       On affiche une alerte"""
       print("Alerte ! Il n'y a pas d'attribut {} ici !".format(nom))

    def _getSalaire(self):
       """Méthode permettant d'obtenir le salaire"""
        print("Affichage du salaire de " + self.nom)
        return self._salaire

    def _setSalaire(self,nvxSalaire):
       """Méthode permettant de modifier le salaire"""
        print("/!\ modification du salaire de " + self.nom)
        self._salaire=nvxSalaire

   /*** format de la propriété nom_propriete = property(methode_accesseur, methode_mutateur,
    methode_suppression, methode_aide) ***/

   salaire=property(_getSalaire,_setSalaire)

   /*** Les méthodes de conteneur ***/
    def __getitem__(self, index):
        """Cette méthode spéciale est appelée quand on fait objet[index]
        Elle redirige vers self._dictionnaire[index]"""
        return self._dictionnaire[index]

    def __setitem__(self, index, valeur):
        """Cette méthode est appelée quand on écrit objet[index] = valeur
        On redirige vers self._dictionnaire[index] = valeur"""
        
        self._dictionnaire[index] = valeur      



   /***Méthodes mathématiques ***/
   def __gt__(self, autre_salaire):
        """Test si self > autre_salaire"""
          return "{} a un salaire supérieur".format(self.nom) 
   def __le__(self, autre_salaire):
        """Test si self < autre_salaire"""
          return "{} a un salaire supérieur".format(self.nom) 

   /***Méthodes spéciales utiles à pickle ***/
   def __getstate__(self):
        """Renvoie le dictionnaire d'attributs à sérialiser"""
        dict_attr = dict(self.__dict__)
        dict_attr["attribut_temporaire"] = 0
        return dict_attr
Terminologie :

methode_mutateur=Les mutateurs permettent de le modifier.
methode_suppression=la méthode appelée quand on souhaite supprimer l'attribut ;
methode_aide =la méthode appelée quand on demande de l'aide sur l'attribut.
sérialiser =Mettre sous forme binaire des données et de les écrire dans un fichier. désérialiser =Extraire des données dans un fichier sous forme binaire .

Les Méthodes spéciales :

Méthode spéciale Résumé
__getattr__ Python recherche l'attribut et, s'il ne le trouve pas dans l'objet et si une méthode__getattr__existe, il va l'appeler en lui passant en paramètre le nom de l'attribut recherché, sous la forme d'une chaîne de caractères.
__setattr__ Si on écrit objet.nom_attribut = nouvelle_valeur, la méthode spéciale__setattr__sera appelée
__delattr__ supprimer un attribut de l'objet, en faisantdel objet.attribut
__repr__ Elle affecte la façon dont est affiché l'objet quand on tape directement son nom
__del__ méthode qui va être appelée au moment de la destruction de l'objet.
__str__ méthode spéciale,__str__, spécialement utilisée pour afficher l'objet avec print
__len__ est appelée quand on souhaite connaître la taille d'un objet conteneur
__getitem__ sert quant à définir quand on écrit objet[index];
__setitem__ sert quant à définir quand on écrit objet[index] = valeur;
__delitem__ sert quant à définir quand on écrit del objet[index];
__eq__ Opérateur d'égalité (equal). Renvoie True si self et objet_a_comparersont égaux, False sinon.
(def __eq__(self, objet_a_comparer):)
__ne__ Différent de (non equal). Renvoie True si self et objet_a_comparer sont différents, False sinon.
__gt__ Teste si self est strictement supérieur (greater than) à objet_a_comparer.
__ge__ Teste si self est supérieur ou égal (greater or equal) à objet_a_comparer.
__lt__ Teste si self est strictement inférieur (lower than) à objet_a_comparer.
__lt__ Teste si self est inférieur ou égal (lower or equal) à objet_a_comparer.
__getstate__ est appelée au moment de sérialiser l'objet
__setstate__ est appelée au moment de désérialiser l'objet.
