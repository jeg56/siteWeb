On rapporche nos données à présenter de la carte.

On effectue une jointure entre nos 2 sources par leur point commun : le code du département

# Jointure gauche
departements@data <- left_join(departements@data, SOURCE, by = c("CODE_DEPT" = "Code"))

# Vérification que la jointure c'est bien passé
departements@data[is.na(departements@data$ValeurARepresenter),]

Préparation des intervalles de regroupement avec leurs couleurs.

On effectue un découpage par quantile sur 5 niveaux - soit de 20% en 20% Vous pouvez effectuer un découpage personnalisé avec classAuto.d
Puis dans un second temps je recupère 5 couleurs

# Création des classes d'intervalles exploitées par la carte
classAuto <- classIntervals(departements@data$ValeurARepresenter, 5, style="quantile")
# ou classAuto.d <- c(min(departements@data$ValeurARepresenter), 20, 25, 30,35,  max(departements@data$ValeurARepresenter)) 
# si vous faites un découpage manuel
palette <- brewer.pal(n = 5, name = "YlOrRd")

Vous pouvez choisir bien d’autre palette de couleur : 

display.brewer.all() 
brewer.pal.info

On génère notre carte.

On importe les librairies nécessaire ( pour rappel : install.packages(“mon package”) si besoin )

# Rajouter les couleurs que l'on a choisi 
departements@data$Color <- as.character(cut(departements@data$ValeurARepresenter, 
                               breaks = classAuto$brks, 
                                # ou breaks = classAuto.d, si vous etes partis sur un découpage manuel 
                               labels = palette, 
                               include.lowest = TRUE))

# On prépare la légende
legende <- levels(cut(departements@data$ValeurARepresenter, 
                      breaks = classAuto$brks,
                      # ou breaks = classAuto.d, si vous etes partis sur un découpage manuel           
                      include.lowest = TRUE, 
                      right = FALSE))


# On affiche notre carte avec sa légende
plot(departements, col = departements@data$Color, border = "grey")
legend("topleft", legend = legende, fill = palette, cex=0.8, title = "Légende des couleurs")
